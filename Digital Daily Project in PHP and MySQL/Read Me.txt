How to run the News Portal Project using PHP and MySql

1. Download the the zip file
2. Extract the file and copy digitaldaily folder
3.Paste inside root directory(for xampp xamp/htdocs, for wampp/www,fpr lamp var/www/html)
4. Create a database with name digitaldaily
5. Import digitaldaily.sql file(given inside the zip package in sql file folder)
6.run the script http://localhost/digitaldaily (frontend)
7. For admin panel  http://localhost/digitaldaily/admin
